# 00 - Project Overview

# ID Photo AI

## 1. 프로젝트 개요

ID Photo AI는 사용자가 일상 사진 여러 장을 업로드하면  
자동으로 증명사진 규격에 맞는 결과물을 생성하는 AI 기반 시스템이다.

이 프로젝트는 단순한 이미지 필터가 아니라,  
여러 단계의 컴퓨터 비전 파이프라인을 통해  
이미지를 분석하고 변환하는 구조로 설계된다.

---

## 2. 이 프로젝트를 통해 배우는 것

이 프로젝트는 단순히 "AI 모델을 사용하는 것"이 아니라,  
다음과 같은 시스템 설계 능력을 기르는 것을 목표로 한다.

- 웹 서버 구조 이해 (FastAPI)
- 파일 업로드 및 저장 구조 설계
- Job 기반 작업 관리 시스템
- 이미지 처리 파이프라인 설계
- 품질 검사 자동화 로직 구현
- 결과 리포트(report.json) 생성

---

## 3. 핵심 개념 정리

### 3.1 MVP (Minimum Viable Product)

MVP란 "최소 기능 제품"을 의미한다.

완벽한 최종 버전을 처음부터 만드는 것이 아니라,  
핵심 기능만 먼저 구현하여 작동하는 최소한의 시스템을 만드는 전략이다.

이 프로젝트의 MVP 범위는 다음과 같다:

1. 이미지 업로드
2. Job 폴더 생성
3. 얼굴 탐지
4. 기본 품질 검사
5. 얼굴 정렬
6. 배경 단색 합성
7. 크롭 및 리사이즈
8. 결과 이미지 및 report.json 생성

---

### 3.2 Job 시스템

Job이란 하나의 작업 단위를 의미한다.

사용자가 사진 여러 장을 업로드하면  
그 업로드 묶음 전체를 하나의 Job으로 간주한다.

폴더 구조 예시:

data/jobs/<job_id>/
 ├── uploads/
 └── outputs/

이 구조를 사용하는 이유:

- 동시에 여러 사용자가 요청해도 파일이 섞이지 않는다.
- 디버깅이 쉽다.
- 각 작업의 기록을 독립적으로 보관할 수 있다.

---

### 3.3 컴퓨터 비전 파이프라인

본 프로젝트는 다음과 같은 단계로 구성된다.

1. 얼굴 탐지 (Face Detection)
2. 품질 검사 (Quality Check)
3. 얼굴 정렬 (Face Alignment)
4. 배경 제거 및 단색 합성
5. 규격 크롭 (Crop)
6. 이미지 크기 조정 (Resize)
7. 최종 검증 및 리포트 생성

각 단계는 독립적인 모듈로 구현될 예정이다.

---

## 4. 프로젝트 선택 이유

- 실생활에서 활용 가능한 문제 해결
- AI + 서버 + 시스템 설계를 동시에 경험 가능
- 포트폴리오로서의 가치가 높음
- GPU 환경(RTX 3060 Ti)을 활용한 확장 가능성
